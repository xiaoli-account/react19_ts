# Kiro Steering 文件

此目录包含 Kiro 的项目知识库文件（Steering Files）。

## 文件说明

- `project-overview.md` - 项目概述、技术栈、代码规范等核心信息

## Steering 文件的作用

Steering 文件是 Markdown 格式的项目知识库，用于：
- 提供项目上下文信息
- 指导 AI 理解项目结构和规范
- 确保生成的代码符合项目标准

## 配置优先级

Steering 文件的优先级：**Project > Global**

项目级 Steering 文件会覆盖全局配置。

## 更多信息

参考 [Kiro 官方文档](https://kiro.dev/docs/cli/steering/) 了解更多关于 Steering 文件的配置方式。
