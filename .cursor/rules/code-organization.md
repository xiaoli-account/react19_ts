# 代码组织规则

## 代码存放位置

- **代码 < 500 行** → 放入 `src/` 对应目录
- **代码 ≥ 500 行** → 放入 `src_ai/` 对应目录

## 重要说明

- `src_ai/` 是项目正式组成部分，不是临时目录
- 根据代码行数决定存放位置，确保项目结构清晰

## 目录结构

```
src/          - 手动编写的代码（< 500 行）
src_ai/       - AI 生成的代码（≥ 500 行）
  components/ - AI 生成的组件
  pages/      - AI 生成的页面
  services/   - AI 生成的服务
```
